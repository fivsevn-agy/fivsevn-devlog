# Intake

`intake/` generates the public daily intake page:

https://devlog.fivsevn.com/intake/

The page is a lightweight daily reading shelf. It is built automatically by GitHub Actions and currently includes:

- RSS reading sections from `feeds.yml`
- a Wikimedia Commons daily postcard
- a GBIF arthropod field sample

## Files

```
intake/
├── README.md
├── feeds.yml
├── requirements.txt
├── build.py
├── index.html
├── scripts/
│   ├── daily_postcard.py
│   └── daily_field_sample.py
└── data/
    ├── daily_postcard.html
    ├── daily_postcard.json
    ├── daily_field_sample.html
    └── daily_field_sample.json
```

## How it works

`feeds.yml` defines the RSS sections and feed sources.

The daily scripts generate small HTML and JSON fragments:

```
scripts/daily_postcard.py
scripts/daily_field_sample.py
```

`build.py` combines those fragments with RSS items and writes:

```
intake/index.html
```

## GitHub Actions

The page is generated by:

```
.github/workflows/build-news.yml
```

The workflow installs dependencies, runs the daily scripts, runs `build.py`, and commits the generated files back to the repository.

Generated files include:

```
intake/index.html
intake/data/daily_postcard.html
intake/data/daily_postcard.json
intake/data/daily_field_sample.html
intake/data/daily_field_sample.json
```

## Manual build

From the repository root:

```
pip install -r intake/requirements.txt
python intake/scripts/daily_postcard.py
python intake/scripts/daily_field_sample.py
python intake/build.py
```

## Editing notes

Edit `feeds.yml` to change RSS sources or sections.

Edit `build.py` to change the page structure or rendering.

Edit scripts in `intake/scripts/` to change the daily generated modules.

Do not hand-edit `index.html` or files in `intake/data/` as source files. They are generated outputs and may be overwritten by GitHub Actions.

## Boundary

`intake/` is a current-facing generated page, not a permanent archive or canonical content axis.
